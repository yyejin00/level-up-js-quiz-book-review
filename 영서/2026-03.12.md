
## 📌 오늘의 주제 : 객체 속성 메서드

## 🔥 한 줄 핵심
- JS에서 데이터는 대부분 객체로 흐르기 때문에, 객체 메서드를 알면 데이터 가공, 상태 관리, 버그 방지 모두 수월해진다.

### ✏️ 키워드 정리

| 메서드          | 설명 | 예시      |
| -------- | -------------- | ------------- |
| **Object.keys(obj)s**      | 키 배열 반환      | ['name', 'age', 'city']  |
| **Object.values(obj)** | 값 배열 반환   | ['Alice', 30, 'Seoul']              |
| **Object.entries(obj)**      | [키, 값] 쌍 배열 반환    | [['name','Alice'], ['age',30], ['city','Seoul']] |

```js
const user = { name: 'Alice', age: 30, city: 'Seoul' };

Object.keys(user);    // ['name', 'age', 'city']
Object.values(user);  // ['Alice', 30, 'Seoul']
Object.entries(user); // [['name','Alice'], ['age',30], ['city','Seoul']]

// 실전 - 객체 순회
Object.entries(user).forEach(([key, value]) => {
console.log(`${key}: ${value}`);
});

```
