## 📌 오늘의 주제 : 이터러블(Iterable), 이터레이터(Iterator)

## 🔥 한 줄 핵심

- Iterable은 **`Symbol.iterator`** 를 가진 객체이고,
  Iterator는 **`next()`** 메서드를 가진 객체이다.

### ✏️ 키워드 정리

|                | 설명                           |
| -------------- | ------------------------------ |
| **이터너블**   | 반복 가능한 데이터             |
| **이터레이터** | 실제로 값을 하나씩 꺼내는 도구 |

```js

//흐름

Array
 ↓
Symbol.iterator()
 ↓
Iterator 생성
 ↓
next()
 ↓
값 하나씩 반환

```
