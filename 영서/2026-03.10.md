## 📌 오늘의 주제 : Symbold의 이해

## 🔥 한 줄 핵심

- Symbol은 고유하고 변경 불가능한 값을 생성하는 JavaScript의 원시 타입으로, 주로 객체 프로퍼티의 네임스페이스 충돌을 방지하기 위해 사용된다.

### ✏️ Symbol이 추가된 이유

1. 객체 property 충돌 방지
2. 라이브러리 내부 key 보호
3. JavaScript 내부 기능 확장

```js
const user = {
  id: 1,
};

// 다른 코드
user.id = 'abc';

//결과
user.id; // "abc"
```

### 🤖 Symbol 사용법

```js
const id = Symbol("id");

const user = {
  name: "Tom",
  [id]: 123
  const id = Symbol("id");
}

user.id = '여기요~';

//결과
user;
// {id: "여기요~", name: "Tom", Symbol(id): 123}

```
