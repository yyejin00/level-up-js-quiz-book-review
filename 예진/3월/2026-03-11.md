# 📌 오늘의 주제 : 객체 생성 / 객체 속성 다루기
 📖 page 111 ~ 124 퀴즈 3번 문제

## 🔥 한 줄 핵심
객체 표기법,변수에 대해 알아가기
### ✏️ 키워드 정리
- 객체 전달하기 : 객체의 참조 특성 때문에 가능한 일
  객체동결 / 객체를 직접 복사하는 방법으로 참조때문에 일어나는 현상 막기
- 객체 복사 : 깊은 복사`JSON.stringify`,`JSON.parse` / 얕은 복사`Object.assign`
- 이터레이터 : 이터러블 프로토콜, 이터레이터 프로토콜
  이터러블 프로토콜 : 특정 문법에서 객체가 순회 가능 하도록 정의하는 규칙.
              `Symbol.iterator`
               이터러블 객체
  이터레이터 프로토콜 : 객체 반복 과정 중, 다음 값을 반환하는 방식과 반복이 조료되었는지의 여부를 정의.
              Next / value / done / 이터레이터 객체
  => for문 순회가 그냥 일어나는 게 아님.
  이터러블 이터레이터 : 이터레이터 프로토콜 + 이터러블 프로토콜 모두 정의 된 객체
- 제너레이터 : 복잡한 이터레이터 구현을 쉽게 한다.
            제너레이터 객체 = 이터러블 이터레이터
            `function*`
  
- 키 기반 컬렉션 : 키를 기준으로 데이터를 처리
  Map : 키와 값의 쌍을 저장하는 자료구조 /
  Set : 값만 저장. 집합연산 가능.
  WeakMap,WeakSet :데이터가 객체 안에서만 참조된다면 삭제해서 메모리 누수에 도움을 준다.
