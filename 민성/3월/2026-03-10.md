## 📌 오늘의 주제 : React Server Components의 이해

## 🔥 한 줄 핵심

- 클라이언트 사이드 번들 크기를 줄이고 성능을 최적화할 수 있다.

### ✏️ 키워드 정리

- server compoents : 서버컴포넌트가 있으면 클라이언트 컴포넌트도 있는 법
  서버에서 직접 데이터를 페치하고 렌더링하여 클라이언트 번들 사이즈를 줄이는 기술

## ✨ 딱 하나만 기억한다면? 코드 조각이나 이론 중 가장 중요했던 핵심 포인트 하나만 기록합니다.

"컴포넌트 트리의 일부는 서버에서 실행하고, 일부는 브라우저에서 실행하자"

Server Component: 데이터 로딩, DB 접근, 무거운 연산 담당(브라우저로 코드 전송 X)
Client Component: 클릭, 입력, 상태 관리 같은 인터랙션 담당(브라우저로 코드 전송 O)
