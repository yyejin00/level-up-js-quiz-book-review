
# 📚 QuizBook (퀴즈북)
매일 아침, 성장을 위한 짧고 굵은 기록 > 완벽함보다는 '꾸준함'에 집중하며, 오늘 배운 내용을 퀴즈처럼 명확하게 정리하는 공간입니다.

## 🎯 나만의 규칙 (Ground Rules)
지치지 않고 오래 지속하기 위해 아래의 세 가지 규칙을 지킵니다.

⏰ 평일 오전 8시 20분. 상쾌한 아침 시간을 활용해 공부하기

📝 작성할 내용 : 오늘 공부한 내용을 누구나 읽기 쉽게 짧게 작성하기

⏳ 1시간의 마법 : 정리에 1시간 이상 쏟지 않기! (기록의 부담을 줄이고 습관 형성에 집중합니다. ⚡️)

### 🛠 공부 기록 템플릿 (Sample)
기록할 때 아래 형식을 참고하면 더 빠르게 작성할 수 있어요.

📅 제목: YYYY-MM-DD
오늘의 주제: (예: React Server Components의 이해)

한 줄 핵심: (예: 클라이언트 사이드 번들 크기를 줄이고 성능을 최적화할 수 있다.)

내용 : 키워드 / 흐름 중심으로 작성
